# ts-crud-api

## Need to Install

- NodeJS
- PostgreSQL

## How to Run

1. Create database using `createdb <DATABASE NAME>`
2. Create `.env` file to add db credential
   POSTGRES_HOST=<HOSTNAME>
   POSTGRES_PORT=<PORT>
   POSTGRES_USER=<USER>
   POSTGRES_PASSWORD=<PASSWORD>
   POSTGRES_DB=<DATABASE_NAME>
3. Run `yarn install`
4. Run `yarn start-dev`
